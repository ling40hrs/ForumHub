- Centralized AI workflow in `CLAUDE.md` (5-phase role-gated process)
- Role-gate and quality-gate AI skills (`.mimocode/skills/`)
- Cross-layer coordination template and issue templates
- API contract (`docs/api-contract.md`)
- CI pipeline (PHP lint, frontend lint + build)
- Linter configs (PHP CodeSniffer, ESLint)
- One-command setup (`bin/setup`)
- `.editorconfig`, `.gitattributes`, `Makefile`, VS Code config
- Docker Compose for backend services
- Seed script for test data
- Pre-commit hook for quality enforcement
